Text Formatting tags

b & strong

b: This tag is used to make text bold without conveying any extra importance. strong: This tag also makes text bold but indicates that the text is of strong importance

This is bold text and this is important text.

i & em

i: This tag italicizes text without implying any added emphasis. em: This tag italicizes text and suggests that the text has added emphasis.

This is italic text and this is emphasized text.

u

underlines text, which can be useful for stylistic purposes.

This is underlined text.

mark

highlights text, typically indicating that it is relevant or important in the context

This is a highlighted text that needs attention.

small

tag is used to display text in a smaller font size, often used for legal disclaimers or side notes

This is smaller text than the surrounding text.

del & ins

del: This tag indicates deleted text, typically shown with a strikethrough. ins: This tag represents inserted text, usually underlined.

The price was $20.00 and is now $15.00.

sub & sup

sub: This tag is used for subscript text, which appears lower than the baseline (e.g., chemical formulas). sup: This tag is used for superscript text, which appears higher than the baseline (e.g., exponents).

Water is represented as H2O and the square of x is x2.

code

used to define a piece of computer code. The text inside is typically displayed in a monospace font.

Use the command git status to check the status of your repository.

blockquote

used for quoting large sections of text, often displayed as indented text

“The only limit to our realization of tomorrow is our doubts of today.” - Franklin D. Roosevelt
